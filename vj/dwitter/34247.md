## Analyse visuelle

### Principe : sphère en espace négatif

La sphère n'est **pas dessinée** — elle est **effacée** (`clearRect`) dans un fond de lignes.

1. **Fond** : `x.fillRect(i-8e3, 0, 0.2, 2e3)` dessine ~2000 lignes verticales ultra-fines
   (0.2px de large) couvrant x ∈ [1, 1999] — crée une texture de grille subtile
2. **Sphère** : chaque point visible appelle `x.clearRect(cx, cy, 6, 6)` — crée un trou 6×6
   dans le fond, révélant la transparence (noir de la page derrière le canvas)
3. **Résultat** : sphère en trous noirs sur fond de lignes colorées

### Géométrie sphérique

- **10 000 itérations**, mais seules ~6 000 génèrent des points valides (Z ∈ [-1, 1])
- `a = i³ - t*2` : exposant cubique → spirale très dense, bonne couverture de surface
- Pas de risque de N = NaN (le culling NaN > 0 = false protège clearRect)

### Deux axes de rotation

| Rotation | Axe | Angle | Vitesse |
|----------|-----|-------|---------|
| Rotation 1 (Y–Z) | Axe X | `b = t/3` | Lente (periode ~19s) |
| Rotation 2 (X–W) | Axe Y | `t` | Rapide (période ~6s) |

La rotation 1 incline la sphère dans le plan Y–Z :
- `W = sin(b)·Y + cos(b)·Z` (nouveau Z pour la rotation 2)
- `Y_screen = Y·cos(b) - Z·sin(b)` (composante Y à l'écran)

La rotation 2 tourne autour de Y :
- `X_screen = X·sin(t) + W·cos(t)` (projeté sur X écran)
- Culling : `X·cos(t) - W·sin(t) > 0` (face avant uniquement)

## Traduction Python prévue

### Particularités à adapter

- **Pas de clear global** : en Python/PIL, recréer le fond de lignes à chaque frame
- **clearRect → trou dans fond** : dessiner le fond d'abord, puis "effacer" les points sphère
  avec des patches noirs (ou couleur de fond) de 6×6px
- **Deux rotations** à appliquer successivement
- **Double rôle du for** (corps + post-step) → boucle Python simple : calculer puis dessiner
- **10 000 iterations** : séquentiel comme l'attractor, mais 40% inutiles (Z > 1)

### Adaptation couleur

- Remplacer le fond monochrome (lignes blanches sur noir) par des lignes de couleur palette
- La sphère reste en espace négatif (trous dans le fond)
- Possibilité de colorer les trous en couleur palette secondaire plutôt que noir pur
