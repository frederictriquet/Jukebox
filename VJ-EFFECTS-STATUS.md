# VJing Effects (33 implémentés)

| Effet | Description |
|-------|-------------|
| `pulse` | Cercles expansifs sur beats |
| `strobe` | Stroboscope modulé par les aigus |
| `fft_bars` | Barres FFT verticales colorées |
| `fft_rings` | Anneaux concentriques FFT |
| `bass_warp` | Déformation polygonale par basses |
| `particles` | Particules colorées rythmiques |
| `flow_field` | Champ de flux Perlin/Simplex |
| `explosion` | Explosion de particules sur beats forts |
| `starfield` | Champ d'étoiles 3D avec perspective |
| `kaleidoscope` | Motifs kaléidoscopiques symétriques |
| `lissajous` | Courbes de Lissajous modulées |
| `tunnel` | Tunnel infini avec profondeur |
| `spiral` | Spirale animée colorée |
| `radar` | Balayage radar circulaire avec blips |
| `grid` | Grille 2D déformée par ondes sur kicks |
| `fractal` | Fractales Julia animées (GPU) |
| `plasma` | Plasma coloré ondulant (GPU) |
| `wormhole` | Trou de ver avec distorsion spirale (GPU) |
| `voronoi` | Diagramme de Voronoï animé (GPU) |
| `metaballs` | Métaballs fluides (GPU) |
| `fire` | Flammes animées |
| `water` | Ondulations d'eau sur beats |
| `aurora` | Aurore boréale ondulante |
| `smoke` | Simulation de fumée avec turbulence |
| `lightning` | Éclairs ramifiés sur beats |
| `wave` | Vagues sinusoïdales fluides |
| `neon` | Formes néon pulsantes |
| `vinyl` | Sillons de vinyle rotatifs |
| `chromatic` | Aberration chromatique RGB (post-process) |
| `pixelate` | Pixelisation dynamique (post-process) |
| `feedback` | Traînées avec décroissance (post-process) |
| `timestretch` | Ralenti/accélération selon énergie (post-process) |
| `bloom` | Glow neon audio-réactif (final pass) |
